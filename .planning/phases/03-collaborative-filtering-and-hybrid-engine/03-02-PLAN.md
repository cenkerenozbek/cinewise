def build_similarity_index(tfidf_matrix, top_n: int = 50) -> np.ndarray:
    N = tfidf_matrix.shape[0]
    effective_top_n = min(top_n, N - 1)
    top_indices = np.zeros((N, effective_top_n), dtype=np.int32)
    for i in range(N):
        sims = cosine_similarity(tfidf_matrix[i], tfidf_matrix).flatten()
        sims[i] = -1.0  # exclude self
        top_indices[i] = np.argpartition(sims, -effective_top_n)[-effective_top_n:]
    return top_indices

def save_artifacts(vectorizer, tmdb_ids, top_indices, artifacts_dir):
    os.makedirs(artifacts_dir, exist_ok=True)
    joblib.dump(vectorizer, os.path.join(artifacts_dir, "tfidf_vectorizer.joblib"))
    joblib.dump({"tmdb_ids": tmdb_ids, "top_indices": top_indices},
                os.path.join(artifacts_dir, "similarity_index.joblib"))

    **Script purpose:** Download and process MovieLens-20M `links.csv` and `ratings.csv`, map MovieLens movie IDs to TMDB IDs, filter ratings to like (>= 4.0) and dislike (<= 2.0), insert as synthetic interactions for a bounded subset of users.

    **Implementation details:**

    1. Use `csv.reader` (not pandas) to read CSV files to avoid adding a dependency. CSV files are expected at paths provided via env var `MOVIELENS_DIR` (default: `/data/movielens`).

    2. Load `links.csv` columns: `movieId,imdbId,tmdbId`. Build mapping `ml_movie_id -> tmdb_id`. Skip rows where tmdbId is empty or non-numeric (NaN handling).

    3. Query MongoDB `movies` collection for all tmdb_ids to get the set of TMDB IDs that exist in our DB: `existing_tmdb_ids = set()`. Only insert interactions for movies we actually have.

    4. Load `ratings.csv` columns: `userId,movieId,rating,timestamp`. Filter:
       - rating >= 4.0 -> action = "like"
       - rating <= 2.0 -> action = "dislike"
       - 2.0 < rating < 4.0 -> skip (ambiguous)

    5. Limit to first 1000 unique MovieLens user IDs encountered (env-configurable via `SEED_USER_LIMIT`, default 1000). Namespace as `seed_user_{userId}`.

    6. Idempotent: before inserting, delete all existing interactions where `user_id` starts with `seed_user_`: `await db.interactions.delete_many({"user_id": {"$regex": "^seed_user_"}})`.

    7. Batch insert using `insert_many` in chunks of 5000 for efficiency.

    8. Log statistics: "Seeded X interactions for Y users across Z movies".
