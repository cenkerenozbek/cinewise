   - `build_similarity_index(tfidf_matrix, top_n: int = 50) -> np.ndarray` function:
     - `N = tfidf_matrix.shape[0]`
     - `effective_top_n = min(top_n, N - 1)` (handle small corpora)
     - `top_indices = np.zeros((N, effective_top_n), dtype=np.int32)`
     - Iterate row by row (memory-safe): `for i in range(N):`
       - `sims = cosine_similarity(tfidf_matrix[i], tfidf_matrix).flatten()`
       - `sims[i] = -1.0` (exclude self -- set to -1 to ensure self is never selected)
       - `top_indices[i] = np.argpartition(sims, -effective_top_n)[-effective_top_n:]`
     - Return `top_indices`

   - `save_artifacts(vectorizer, tmdb_ids: list[int], top_indices: np.ndarray, artifacts_dir: str) -> None` function:
     - `os.makedirs(artifacts_dir, exist_ok=True)`
     - `joblib.dump(vectorizer, os.path.join(artifacts_dir, "tfidf_vectorizer.joblib"))`
