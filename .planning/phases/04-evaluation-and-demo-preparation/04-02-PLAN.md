**Pure functions (testable without DB):**

`precision_at_k(recommended_ids: list[int], relevant_id: int, k: int = 10) -> float` — Returns 1.0 if relevant_id in recommended_ids[:k], else 0.0.

`compute_ndcg_at_k(recommended_ids: list[int], relevant_id: int, k: int = 10) -> float` — Uses `sklearn.metrics.ndcg_score`. Builds y_true (1.0 at position of relevant_id, 0.0 elsewhere) and y_score (descending rank scores: k, k-1, k-2, ...). Both are 2D arrays of shape (1, k). Returns 0.0 if relevant_id not in top-k.

`build_leave_one_out_test_set(interactions: list[dict], min_likes: int = 5, max_users: int = 500) -> list[tuple[str, int, list[int]]]` — Groups interactions by user_id, filters to action=="like", excludes users with fewer than min_likes likes, sorts each user's likes by `updated_at` ascending, holds out the last one as ground truth, returns (user_id, held_out_movie_id, training_movie_ids) tuples. Random-shuffles and caps at max_users.

`async def ensure_user_exists(db, email: str, password: str) -> str`:
- Check if user exists: `await db.users.find_one({"email": email})`
- If exists, return `str(user["_id"])`
- If not, hash password with passlib bcrypt (import from `passlib.hash import bcrypt as passlib_bcrypt`; use `passlib_bcrypt.hash(password)`) and insert new user document `{"email": email, "hashed_password": hashed}`. Return the string `_id`.
- Log whether created or found.

`async def reset_demo(db, dry_run: bool = False)`:
1. Ensure demo_returning exists, get user_id
2. Ensure demo_coldstart exists, get user_id
3. Delete ALL interactions for demo_returning user_id: `await db.interactions.delete_many({"user_id": returning_user_id})`
4. Delete ALL interactions for demo_coldstart user_id: `await db.interactions.delete_many({"user_id": coldstart_user_id})`
5. Check which canonical TMDB IDs actually exist in our movies collection:
   ```python
   existing = await db.movies.find(
       {"tmdb_id": {"$in": CANONICAL_LIKE_TMDB_IDS}},
       {"tmdb_id": 1}
   ).to_list(length=None)
   valid_tmdb_ids = [d["tmdb_id"] for d in existing]
   ```
   If fewer than 5 valid IDs, fall back to top-6 rated movies:
   ```python
   if len(valid_tmdb_ids) < 5:
       fallback = await db.movies.find(
           {"rating": {"$ne": None}}, {"tmdb_id": 1}
       ).sort("rating", -1).limit(6).to_list(length=None)
       valid_tmdb_ids = [d["tmdb_id"] for d in fallback]
   ```
6. Re-seed canonical likes for demo_returning:
   ```python
   from datetime import datetime, timezone
   for tmdb_id in valid_tmdb_ids:
       await db.interactions.update_one(
           {"user_id": returning_user_id, "movie_id": tmdb_id},
           {"$set": {"user_id": returning_user_id, "movie_id": tmdb_id,
                      "action": "like", "updated_at": datetime.now(timezone.utc)}},
           upsert=True,
       )
   ```
7. Print confirmation: number of likes seeded, both account emails.

`async def cleanup_uat_accounts(db, uat_prefix: str, dry_run: bool = False)`:

2. Collect their _ids as strings
3. Delete their interactions: `await db.interactions.delete_many({"user_id": {"$in": uat_ids}})`
4. Delete the user accounts: `await db.users.delete_many({"email": {"$regex": f"^{uat_prefix}"}})`
5. Delete their preferences: `await db.user_preferences.delete_many({"user_id": {"$in": uat_ids}})`
6. Print confirmation: number of UAT accounts and interactions deleted.

If `dry_run` is True, skip all write operations and only print what WOULD happen.
